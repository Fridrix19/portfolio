# Portfolio Website

Современный сайт-портфолио в виде лендинга с черно-белым дизайном.

## Особенности

- ✅ Чистый HTML/CSS/JavaScript
- ✅ Черно-белая цветовая схема
- ✅ Кастомный курсор (точка)
- ✅ Плавные анимации
- ✅ Адаптивный дизайн
- ✅ Фильтрация проектов по категориям (UX/UI, WEB-разработка, Анимация)
- ✅ Отдельные страницы для каждого проекта

## Структура проекта

```
port/
├── index.html      # Главная страница
├── product.html    # Шаблон страницы проекта
├── styles.css      # Все стили
├── script.js       # Основная логика (курсор, фильтры)
├── product.js      # Данные проектов
└── README.md       # Документация
```

## Использование

Просто откройте `index.html` в браузере. Все файлы связаны локально, дополнительные зависимости не требуются.

## Кастомизация

### Добавление нового проекта

1. Добавьте новую карточку проекта в `index.html` в секции `.projects-container`
2. Добавьте данные проекта в объект `products` в файле `product.js`
3. Укажите правильный `id` в ссылке проекта

### Изменение стилей

Все стили находятся в `styles.css`. Основные переменные цветов определены в `:root`.

### Изменение анимаций

Анимации определены в конце файла `styles.css` с помощью `@keyframes`.

## Браузерная поддержка

Сайт работает во всех современных браузерах, поддерживающих CSS Grid, Flexbox и ES6 JavaScript.
